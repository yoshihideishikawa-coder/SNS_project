# ターゲットユーザー

## メインペルソナ：Alex (20代・男性・アメリカ在住)

### 基本情報
- **職業**: 大学生 または 若手社会人
- **趣味**: 日本のアニメ（Naruto, Jujutsu Kaisen, Your Name等）、写真撮影
- **訪日経験**: 初めて または 2回目
- **使用言語**: 英語（日本語は挨拶程度）

### 1日の流れ（旅行中）
- 朝ホテルで「今日はどこ行こうかな」とスマホを見る
- 綺麗な写真を見つけて「ここカッコいい！」となる
- 電車と徒歩でその場所へ向かう
- 現地でアニメと同じ構図で写真を撮る
- 撮った写真をSNSにアップする

### 課題・ペイン
- 「Google Mapsで検索しても、英語だと正確な場所が出てこないことがある」
- 「アニメのシーンと実際の場所が一致するか不安」
- 「効率よく回りたいけど、ルートを組むのが面倒」

### 求めているもの
- **Visual Inspiration**: 「わぁ、すごい！」となる写真
- **Certainty**: 「ここに行けば間違いない」という確信と正確な地図
- **Social Validation**: 「俺もあそこ行ったぜ」と自慢したい承認欲求

## サブペルソナ：Sarah (30代・女性・フランス在住)
- **趣味**: ジブリ映画、日本の古い街並み、歴史
- **特徴**: 「映え」よりも「雰囲気」や「ストーリー」を重視する
- じっくりと計画を立てて、失敗のない旅をしたい慎重派
