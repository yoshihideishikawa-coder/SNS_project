# Seichi Route (仮)

## 概要
「そのシーン、どこ？」を一瞬で解決し、誰もが迷わず聖地巡礼できる、訪日外国人向けビジュアルSNS。

## 解決する課題
- 日本のアニメ・漫画・文化に興味があるが、具体的な場所や行き方がわからない
- ガイドブックや文字情報だけではイメージが湧かず、計画が立てにくい
- 「映える」写真が撮りたいが、どこでどう撮ればいいかわからない

## ターゲットユーザー
- **訪日外国人観光客**（旅行前〜旅行中）
- アニメ、マンガ、日本史、日本文化のファン
- 写真を撮るのが好きで、SNSでシェアしたい層

## 主要機能
- **Visual Exploration**: 写真メインの直感的な聖地探し（タイムライン/グリッド切替）
- **Auto Route Generation**: 写真をアップするだけで、位置情報から自動で巡礼ルートを生成
- **Seichi Database**: 正確な聖地情報のマスタ管理
- **Navigation**: 写真の場所へ迷わず行ける地図・ナビゲーション
