# ユーザーフロー

## 1. 閲覧・計画フロー（Before Trip / During Trip）

1. **アプリ起動**
   - ホーム画面が表示される（Visual Juicyな写真一覧）
   - 必要に応じて「タイムライン表示」と「グリッド表示」を切り替える

2. **探索・発見**
   - 気になる写真をタップする
   - その場所の詳細と、前後のスポットを含めた「ルート」が表示される

3. **アクション**
   - 「Save（保存）」ボタンを押して、行きたいリストに入れる
   - 「Go（ナビ開始）」ボタンを押して、地図アプリ等を起動する

## 2. 投稿フロー（After Trip / During Trip）

1. **投稿開始**
   - 「＋」ボタンをタップ
   - カメラロールから、その日に撮った聖地写真を複数枚選択する

2. **自動生成**
   - アプリが写真のExif情報（位置情報・撮影日時）を解析
   - 聖地DBと照合し、「あなたが回ったルート」を自動生成して提示
   - （位置情報がない写真は、手動でスポット検索して紐付け）

3. **確認・公開**
   - ルートとスポットがあっているか確認
   - タイトルやコメント（任意）を入力
   - 「Post」ボタンで公開

## 3. オンボーディング
1. アプリ初回起動
2. 興味のあるジャンル/作品を選択（Naruto, Ghibli, History...）
3. ホーム画面がパーソナライズされる
