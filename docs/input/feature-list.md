# 機能一覧

## 凡例
- 🔴 Must Have（MVP必須）
- 🟡 Should Have（MVP後優先）
- 🟢 Nice to Have（将来対応）

## 1. 閲覧・探索 (Discovery)

| 機能名 | 説明 | 優先度 | 備考 |
|--------|------|--------|------|
| **タイムライン表示** | Instagram風の縦スクロール写真フィード | 🔴 | デフォルト表示 |
| **グリッド表示** | Pinterest風のタイル状写真一覧 | 🔴 | 切り替え可能 |
| **表示切り替え** | タイムラインとグリッドをワンタップで切り替える | 🔴 | |
| **聖地詳細表示** | 写真、スポット名、作品名、地図を表示 | 🔴 | |
| **ルート表示** | スポット間の移動順序、地図上の線を可視化 | 🔴 | |
| **検索・フィルタ** | 作品名、エリア（Tokyo, Kyoto等）での絞り込み | 🟡 | |

## 2. 投稿・作成 (Creation)

| 機能名 | 説明 | 優先度 | 備考 |
|--------|------|--------|------|
| **写真アップロード** | 複数枚の写真を一括選択 | 🔴 | |
| **Exif解析** | 写真の位置情報・日時から順序と場所を特定 | 🔴 | |
| **聖地DB照合** | 位置情報をもとに近くの聖地候補を提示・紐付け | 🔴 | マスタ利用 |
| **ルート自動生成** | 選択されたスポットを時系列で繋いでルート化 | 🔴 | |
| **手動修正** | 自動判定が間違っている場合にスポットを修正 | 🔴 | |

## 3. アカウント・その他 (Account)

| 機能名 | 説明 | 優先度 | 備考 |
|--------|------|--------|------|
| **ユーザー登録/ログイン** | メール、Apple, Google等 | 🔴 | |
| **マイページ** | 自分の投稿、保存したルートの確認 | 🔴 | |
| **いいね/保存** | 気に入った投稿へのリアクション | 🔴 | |
| **多言語対応** | 英語以外の言語対応 | 🟢 | MVPは英語のみ |

## 4. 管理機能 (Admin)

| 機能名 | 説明 | 優先度 | 備考 |
|--------|------|--------|------|
| **聖地DB管理** | 聖地スポット（位置情報、作品情報）の登録・編集 | 🔴 | これがないと始まらない |
