# プロダクト提案：訪日外国人向け聖地巡礼SNS

## 元のアイデア
- **ターゲット**: 日本文化（アニメ・漫画・歴史など）に関心がある訪日外国人観光客
- **課題**: 「どこに行けばいいか迷う」「効率的に回りたい」「同じ趣味の人と繋がりたい」
- **解決策**: 聖地巡礼に特化したSNSプラットフォーム
- **スコープ**: アニメ・漫画だけでなく歴史なども含む（1&2両方）。旅行前の計画と旅行中のナビゲーション両方に対応（1&2両方）。

## 提案一覧

| | 提案 | コンセプト | 特徴 | 向いている人 |
|---|------|------------|------|--------------|
| **A** | **Seichi Snap**<br>(ビジュアル地図SNS) | **「映え」と「場所」で繋がる**<br>Instagram × Google Maps | ・写真から場所を特定・保存<br>・「このシーンと同じ写真が撮りたい」を叶える<br>・地図上で他の巡礼者の投稿が見える | ・写真を撮るのが好きな人<br>・直感的に行き先を決めたい人 |
| **B** | **Japan Quest**<br>(ゲーミフィケーションSNS) | **旅を「冒険」にする**<br>ポケモンGO × Foursquare | ・作品ごとのスタンプラリー機能<br>・巡礼達成度で称号/バッジ獲得<br>・フレンドと攻略度を競う | ・コンプリート欲がある人<br>・アニメ/ゲームの世界観に浸りたい人 |
| **C** | **Fan Route**<br>(旅程シェアSNS) | **「最高のプラン」を共有する**<br>YAMAP / ヤマレコ の聖地版 | ・先輩ファンの「成功した旅程」をコピーできる<br>・移動手段や所要時間のノウハウ共有<br>・Q&Aコミュニティ機能 | ・失敗したくない慎重派<br>・効率的に多くの場所を回りたい人 |

## 次のステップ
1. 各提案（A/B/C）の詳細を見てください。
2. 「Aがいい」「Bのゲーム要素とCのプラン機能を組み合わせたい」など、感想を教えてください。
3. 選んだ方向性をベースに要件を固めていきます。
