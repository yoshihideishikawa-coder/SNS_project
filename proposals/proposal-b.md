# 提案B：Japan Quest（ゲーミフィケーションSNS）

## コンセプト
**日本旅行をRPGにする。作品ごとの「クエスト」を攻略する冒険SNS**

## 解決する課題
- 「行きたいけど、モチベーションが続かない（移動が大変）」
- 「どこから回ればいいかわからない」
- 「ただ行くだけじゃなくて、達成感がほしい」

## ターゲットユーザー
- **コアファン・オタク層**: 好きな作品のためなら労力を惜しまない人
- **ゲーマー気質**: コンプリートやレベル上げが好きな人
- **コミュニティ志向**: 仲間と協力したり競ったりしたい人

## 主要機能
1. **作品別クエスト**: 「『君の名は。』東京コンプリートコース」「『エヴァンゲリオン』箱根防衛戦」などの公式/ユーザー作成クエスト
2. **デジタルスタンプラリー**: GPSチェックインでスタンプやデジタルアイテムを獲得
3. **トラベラーランク**: 巡礼数に応じてランクアップ（例：旅人→巡礼者→聖地守護者）
4. **パーティ機能**: 同じ時期に日本にいるユーザーと一時的なパーティを組み、情報交換や協力ミッションを行う

## プロダクト形態
- **モバイルアプリ（ネイティブ）**: 位置情報と通知、ゲーム的演出が重要になるため

## ビジネスモデル案（仮）
- **デジタルグッズ課金**: アバター衣装、プロフィール装飾
- **自治体・企業タイアップ**: 「この店に行くと限定アイテムゲット」などの集客キャンペーン
- **特別クエスト**: イベント期間の限定クエスト参加権

## この提案の特徴
- **良い点**: リピート率が高くなる。ユーザーの行動を誘導しやすい（地方創生に繋げやすい）。
- **課題**: コンテンツ制作の工数（クエストを作る手間）。ゲームバランスの調整。

## 想定されるユーザーフロー
1. （旅行前）挑戦したい「クエスト（作品）」を選び、エントリーする
2. （旅行中）クエストの指示に従ってスポットを巡り、チェックインする
3. （完了時）コンプリート画面が表示され、デジタルバッジを獲得。SNSでシェアする。
4. （帰国後）ランキングで自分の順位を確認し、次の旅行の計画を立てる

## 参考になりそうなサービス
- ポケモンGO（位置情報ゲーム）
- Foursquare / Swarm（チェックイン・バッジ）
- ドラゴンクエストウォーク
